Distributed Lock

Sliding Window Rate Limit
ZREMRANGEBYSCORE → ZADD → ZCARD

Virtual Queue (FIFO)
ZADD queue:{event} {timestamp} {userId}
Sorted set keyed by join timestamp. ZRANK gives O(log N) position lookup; top-N users can book.

Read-Through Cache

Atomic Transactions
MULTI → HSET → DEL → INCR → ZREM → EXEC
MULTI/EXEC ensures seat mark, lock delete, counter increment, queue removal all succeed or all fail.
